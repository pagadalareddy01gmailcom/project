# 📊 Olist E-Commerce Sales Analysis & Machine Learning Prediction

This project analyzes the **Olist Brazilian E-Commerce dataset** using **Power BI** for business intelligence and **Machine Learning** for sales prediction. It combines data cleaning, interactive dashboards, and predictive modeling to generate valuable business insights.

---

## 🚀 Project Overview

The project consists of three major parts:

- Data Cleaning & Preprocessing
- Business Intelligence Dashboard (Power BI)
- Machine Learning Model for Sales Prediction

The objective is to help businesses understand customer purchasing behavior, sales performance, and predict future sales using different regression algorithms.

## 🤖 Machine Learning

The notebook builds and compares multiple regression models for predicting sales.

### Models Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- XGBoost Regressor

---

## 📈 Model Evaluation Metrics

Each model is evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

The performance of all models is compared to determine the best-performing algorithm.
